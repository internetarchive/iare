### IARE version 0.3.1.27

If you have problems viewing or scrolling becuae of small screen space,
try clicking the "Unlock" icon at the top-right after it says "Scroll".
Also, not yet mobile friendly.

Features
- Click on WikiSignals in URL list header to bring up mini WikiSignals documentation
- Show WikiSignals inline in URL list
- Click inline display of WikiSignals to pop up detailed Signal data
  - View raw signal details in WikiSignals details popup

Caveats:
- NOT mobile friendly yet!
- "Fetch Signal Data" buttons in inline data are non-functional at the moment
- Sorting by WikiSignals is non-functional at the moment
- Visual metric for Signal Indicator strength, positive or negative, is forthcoming
